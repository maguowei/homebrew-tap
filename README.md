# homebrew-tap

[maguowei](https://github.com/maguowei) 的 Homebrew tap。

## Code Manager

面向 Claude Code 用户的本地桌面配置管理应用 · [项目主页](https://github.com/maguowei/code-manager)

```bash
brew install --cask maguowei/tap/code-manager
```

升级 / 卸载：

```bash
brew upgrade --cask code-manager
brew uninstall --cask code-manager
```

> 应用当前未经过 Apple 公证，cask 安装时会自动移除隔离属性。

Cask 的 `version` / `sha256` 由 [code-manager 仓库](https://github.com/maguowei/code-manager) 的 `update-homebrew-cask` workflow 在每次正式发布后自动更新。
